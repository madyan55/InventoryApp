# سجل التحديثات (Changelog)

هذا الملف يوثق جميع التحديثات والإصدارات الخاصة بمشروع **Madyan Inventory ERP**.

## 📌 الإصدار 1.0.0 – (2026-05-08)
- إطلاق النسخة الأولى من النظام.
- إضافة واجهات رئيسية: تسجيل الدخول، لوحة التحكم، المنتجات، المخازن، الجرد، التقارير.
- إنشاء قاعدة البيانات الأساسية (Products, Warehouses, InventoryChecks, InventoryDetails).
- دعم ثنائي اللغة (عربي/إنجليزي).
- إضافة اختبارات وحدات (Unit Tests) للكائنات الأساسية.
